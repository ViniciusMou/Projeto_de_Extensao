Projeto Loja


Sistema web de loja virtual desenvolvido em grupo durante a faculdade, com foco em prática de front-end e lógica de programação, simulando um sistema simples de e-commerce.

Funcionalidades
Listagem de produtos
Visualização de detalhes de produtos
Simulação de carrinho de compras
Navegação entre páginas
Interface simples de loja virtual
Tecnologias utilizadas
HTML
CSS
JavaScript
Objetivo do projeto

O projeto teve como objetivo praticar a criação de interfaces web, manipulação de dados e estruturação de um sistema simples de e-commerce.

Desenvolvimento em grupo

Projeto realizado em equipe na faculdade, com divisão de tarefas entre interface, estrutura e lógica.

Status

Finalizado (projeto acadêmico)
